# Sistema de Inventario en Python

Sistema de gestión de inventario desarrollado en Python utilizando SQLite y Colorama.
Permite registrar, actualizar, eliminar, buscar productos y generar reportes de bajo stock
mediante un menú interactivo en consola.

## Funcionalidades
- Registro de productos
- Visualización de inventario
- Actualización y eliminación de productos
- Búsqueda por ID, nombre o categoría
- Reporte de bajo stock

## Requisitos
- Python 3
- Biblioteca Colorama
